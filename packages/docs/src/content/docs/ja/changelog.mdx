## v0.22.1

**リリース日:** 2026年2月12日

**リリース日:** 2026年2月10日

- `jump` コマンドにMRU（最近使用した順）ソートを追加（最近使ったワークツリーが上位に表示）

## v0.19.0

**リリース日:** 2026年2月7日

### 追加

## v0.12.0

**リリース日:** 2026年1月18日

### 追加

- **マルチランタイムサポート**: Node.js/npm/npxおよびBun環境でのvibe実行をサポート
- **RuntimeAbstractionLayer**: Node.jsとBun両対応の統一ランタイム抽象化レイヤー

### 変更

### 修正

- ネイティブモジュールのエラーハンドリング改善
- Windows検出ロジックの改善
- 低速接続向けにmise HTTPタイムアウトを拡張
- 各種e2eテストの修正と改善

## v0.9.0

**リリース日:** 2026年1月16日

### 追加

- GNU Coding Standards準拠のCLIオプション
  - `--verbose` / `-V`: 詳細出力を有効化
  - `--quiet` / `-q`: 必須でない出力を抑制
  - `--dry-run`の短いエイリアス`-n`
  - 不明なオプションの検出と有用なエラーメッセージ
- `vibe clean`コマンドの**高速削除（Fast remove）**
  - `.vibe-trash-*`にリネームすることでO(1)の即座なディレクトリ削除
  - バックグラウンドでのクリーンアップによりユーザー体験を向上
  - ベンチマーク: 29秒 → 2秒（342Kファイル、14倍高速化）
  - `[clean] fast_remove`設定オプション（デフォルト：有効）

### 変更

- `errorLog`関数によるヘルプテキストの整形改善

### 修正

- `clean`コマンドのシェルインジェクション防止
- クリーン操作のセキュリティ改善

---

## v0.3.0

**リリース日:** 2026年1月2日

### 追加

- リポジトリセキュリティのための**信頼システム**

- 冪等性のある`vibe start` - 既存のWorktreeで実行すると再利用

### 変更

- 国際ユーザー向けに日本語UIメッセージを英語に置換
- リポジトリルートパスの正規化を改善
