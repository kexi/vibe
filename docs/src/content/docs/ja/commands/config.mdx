`config`コマンドは現在有効な設定を表示します。

## 使い方

```bash
vibe config
```

## 出力

`.vibe.toml`と`.vibe.local.toml`がマージされた設定を表示します：

```bash
$ vibe config
Copy:
  files: [".env", ".env.local"]
  dirs: ["node_modules", ".cache"]

Hooks:
  pre_start: ["echo 'Starting'"]
  post_start: ["pnpm install", "pnpm build"]
  pre_clean: ["git stash"]
  post_clean: ["echo 'Done'"]
```

## ユースケース

### 設定の確認

Worktreeを作成する前に設定が正しくセットアップされているか確認：

```bash
vibe config
vibe start feat/new-feature
```

### マージのデバッグ

`.vibe.toml`と`.vibe.local.toml`がどのようにマージされているか確認：

```bash
# マージ結果を確認
vibe config

# 個別ファイルと比較
cat .vibe.toml
cat .vibe.local.toml
```
