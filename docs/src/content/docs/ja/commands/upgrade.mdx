`upgrade`コマンドはvibeの新しいバージョンを確認し、インストール方法に応じたアップグレード手順を表示します。

## 使い方

```bash
vibe upgrade
```

## オプション

| オプション | 説明                                      |
| ---------- | ----------------------------------------- |
| `--check`  | アップグレード手順を表示せずに更新を確認  |

## 出力

### 最新バージョンの場合

```bash
$ vibe upgrade
vibe 0.7.0+abc1234

You are using the latest version.
```

### 更新がある場合

```bash
$ vibe upgrade
vibe 0.7.0+abc1234

A new version is available: 0.8.0

To upgrade:
  brew upgrade vibe

Release notes:
  https://github.com/kexi/vibe/releases/tag/v0.8.0
```

## インストール方法の自動検出

vibeはインストール方法を自動検出し、適切なアップグレードコマンドを表示します：

| インストール方法 | アップグレードコマンド                              |
| ---------------- | -------------------------------------------------- |
| Homebrew         | `brew upgrade vibe`                                |
| Deno/JSR         | `deno install -A --global jsr:@kexi/vibe`          |
| apt (deb)        | `sudo apt update && sudo apt upgrade vibe`         |
| 開発             | `deno task compile`                                |
| バイナリ         | GitHub Releasesのダウンロードリンクを表示          |

## 使用例

### CIでの更新確認

`--check`フラグを使用して、アップグレード手順を表示せずに更新を確認します：

```bash
$ vibe upgrade --check
vibe 0.7.0+abc1234

A new version is available: 0.8.0

Current: 0.7.0
Latest:  0.8.0
```

### 定期的な更新確認

定期的に更新を確認して最新の状態を保ちます：

```bash
vibe upgrade
```

## ネットワーク要件

このコマンドは最新バージョンを確認するためにJSRレジストリへのネットワークアクセスが必要です。Deno経由でインストールした場合は`--allow-net`権限が有効になっていることを確認してください：

```bash
deno install --global --allow-run --allow-read --allow-write --allow-env --allow-ffi --allow-net jsr:@kexi/vibe
```
