# AGENTS.md

## ブランチ戦略

| ブランチ | 用途 |
|---------|------|
| `main` | リリース用。安定版のみ。 |
| `develop` | 開発用。トピックブランチのマージ先。 |

### ワークフロー

1. `develop`からトピックブランチを作成
2. 作業完了後、`develop`にマージ
3. リリース時に`develop`を`main`にマージ

```
main ────●─────────────────●────
         │                 ↑
develop ─┴──●──●──●──●─────┴────
             ↑  ↑
            feat/a feat/b
```
